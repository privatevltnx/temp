## Setup Instructions

1. **Open the website**: Simply open `index.html` in your web browser
2. **Add showcase images**: Place project images in the `img/` folder with names:
   - `project1.jpg` - `project6.jpg`
   - Recommended size: 400x200 pixels
3. **Customize content**: Edit the HTML files to match your specific content
4. **Deploy**: Upload all files to your web hosting service

## Customization

### Colors
The color scheme can be customized in `css/style.css` by modifying the CSS variables:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --accent-color: #06b6d4;
    /* ... other colors */
}
```

### Content
- Update company information in all HTML files
- Replace placeholder email addresses with your actual contact information
- Modify service descriptions to match your offerings
- Add your actual project images to the showcase

### Supabase Integration
To connect with Supabase for the contact form:
1. Set up a Supabase project
2. Create a table for contact submissions
3. Add Supabase client code to handle form submissions
4. Update the contact form JavaScript in `pages/contact.html`
